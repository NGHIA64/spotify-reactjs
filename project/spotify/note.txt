*** Cách Public App React lên Github page:
- Chi tiết: https://www.youtube.com/watch?v=-c3iwW0KQc4
- Bước 1: Cài npm install gh-pages --save-dev
- Bước 2: Thêm vào phần Script file package.json: 
    {"predeploy": "react-scripts build",
    "deploy": "gh-pages -d build"}
- Bước 3: Thêm file .env vào project reactjs: 

    {PUBLIC_URL ="."}

# Lưu ý: bỏ file .env vào file .gitignore

- Bước 4: Chạy npm run deploy
